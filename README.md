# Игра крестики нолики!

## Описание игры:
Консольная игра для двух игроков на поле 3х3.

## Правила игры:
1. Игроки ходят по очереди.
2. Побеждает тот, кто соберет линию (диагональ) из трех символов (х или 0).
3. Игра заканчивается ничьей при заполнении всех клеток.
